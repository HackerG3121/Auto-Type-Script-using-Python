AUTO TYPE SCRIPT

How to Use:

1. Install Python
2. Open CMD/Terminal in this folder
3. Install required packages:

pip install -r requrededpip.txt

4. Run:

python autotype.py

5. Open any text field
6. Press ENTER
7. Script starts typing automatically
8. Press ESC to stop

Notes:
- Put your text inside scriptfile.txt
- Supports 1000+ lines
- Some apps may require Administrator mode
